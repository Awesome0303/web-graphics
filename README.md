# Графика в Web от Антонова Григория

**Live demo:** https://awesome0303.github.io/web-graphics/

Учебный проект — сайт про графику в Web (WebP, SVG, Canvas и пр.).  
Автор: Антонов Григорий

## Запуск локально

1. Клонировать репозиторий
```
git clone https://github.com/Awesome0303/web-graphics.git
```
2. Открыть `index.html` в браузере или запустить локальный сервер:
```
python -m http.server
```
